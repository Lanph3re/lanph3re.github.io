아래처럼 게시글 목록에 본문 일부가 나오게 하려면
실제로 html 문자열인 게시글의 본문에서 plaintext만 추출하는 작업이 필요하다.
서버단에서 이 작업을 하게 해주고 싶었는데 적당한 모듈을 찾았다.
npm install html2plaintext --save
let h2p = require('html2plaintext');
console.log(h2p('Hello World'));   // Hello World
이미지도 태그도 자동으로 없애주니까 적당히 필요할 때 쓰면 좋을 것 같다.
